# HS Fashion E-Commerce Store

Production-ready MERN luxury fashion storefront built for Hoorain Shakeel.

## Overview
This project is a full-stack luxury fashion e-commerce platform featuring a premium customer storefront, protected admin dashboard, Cloudinary image uploads, WhatsApp ordering, and an interactive 360° product viewing experience.

## Tech Stack

### Frontend
- React
- Vite
- Framer Motion

### Backend
- Node.js
- Express.js
- MongoDB
- Cloudinary

## Features
- Responsive luxury storefront
- Featured Products and New Arrivals
- Product search and filtering
- Admin authentication
- Product CRUD management
- Cloudinary image uploads
- WhatsApp order flow
- Interactive 360° product viewer
- Mobile-friendly design
